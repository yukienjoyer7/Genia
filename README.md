# Genia

Parent repo for the Genia frontend and backend.

## Clone

```bash
git clone --recurse-submodules git@github.com:mohamadarif03/Genia.git
```

If already cloned:

```bash
git submodule update --init --recursive
```

## Run

```bash
docker compose up --build
```

- Frontend: http://localhost:5173
- Backend: http://localhost:8000
